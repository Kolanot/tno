Class LogiServ:Activity

rdf:type
owl:Class
rdfs:comment
An Activity denotes an action that is relevant for the purpose of logistics and provides value for a potential customer. An Activity can be Atomic or Complex. Atomic activities can be used to compose more complex activities. Properties: - An Activity must have a Consumer (at least one) - An Activity must have a Provider (at least one) - An Activity must specify a location (at least one) - An Activity must specify a Time interval or instant (at least one)
